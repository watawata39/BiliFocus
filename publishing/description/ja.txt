BiliFocus は、Bilibili をより少ないノイズで使うためのブラウザ拡張機能です。

おすすめ、コメント、広告、ナビゲーション項目、個人ページのモジュールなどを非表示にできます。さらに、クリーン検索モードを使うと、Bilibili のホームを静かで検索に集中しやすいページにできます。

GitHub でオープンソース公開中：
https://github.com/watawata39/BiliFocus

---

主な機能

* クリーン検索モード
Bilibili のメインページを、よりシンプルな検索ページに変更します。検索バー、アバター、選択した右上ナビ項目だけを残し、ホームおすすめ、検索おすすめ、広告、左上ナビゲーションを非表示にします。

クリーン検索モードは Bilibili のメインページにのみ適用され、拡張機能のポップアップからいつでもオフにできます。

* 背景カスタマイズ
クリーン検索モードでは、内蔵壁紙、単色背景、自分でアップロードした画像から背景を選べます。

* 柔軟な表示制御
Bilibili の各モジュールを、必要に応じて非表示または表示にできます。設定は保存され、すぐにページへ反映されます。

* 右上ナビゲーションの制御
表示する右上ナビ項目を選べます。クリーン検索モードでは、表示中の右上ナビ項目がページ左上に整理され、アバターは右側に残ります。

* 「/」検索ショートカット
/ キーを押すと検索バーにすばやくフォーカスできます。YouTube 風の操作です。

* 多言語インターフェース
ポップアップ UI は、簡体中文、English、日本語に対応しています。

注意

* クリーン検索モードは、インストールまたはアップデート後にデフォルトでオンになります。ポップアップからオフにできます。
* クリーン検索モードは https://www.bilibili.com/ にのみ適用されます。
* BiliFocus は Bilibili の現在の新しい UI 向けに設計されています。

本プロジェクトはもともと個人利用のために開発したものです。私が Bilibili を使い続ける限り、BiliFocus は継続的にメンテナンスします。
